Reviews

This project is an interactive system that allows users to write and share reviews, similar to Yelp or Amazon's product reviews. It uses Parse as its cloud-based data storage web service to create, read, update and delete data. It also uses jQuery to dyanmically alter the content of the page and respond to user interactions. The web page provides users with visual feedback while maing asynchronous requests to the server (i.e., loading wheels).

React-Tac-Toe

In this project I use ReactJS and Node.js to create a game of tic-tac-toe. I use the MVC pattern to seperate the game into three different components. The Model keeps track of the game state and stores the data in an object instantiated from the TicTacToeGame class. The View displays the game to the user and is implemented with React. Finally, the controller acts an an intermediarty between the model object and React view components. It takes user input from the view, calls methods on the model to modify it, and then re-renders the view with the changes.
